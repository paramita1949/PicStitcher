PicStitcher Plus - 更新日志
========================================

v1.5.0 (2025-10-01)
----------------------------------------
【重大更新】
1. 项目迁移到 GitHub 私有仓库
   - 仓库地址: https://github.com/paramita1949/PicStitcher
   - 支持版本控制和协作开发

2. 配置 GitHub Actions 自动打包
   - 推送版本标签自动触发构建
   - 同时提供 PyInstaller 和 Nuitka 两种打包版本
   - 自动创建 Release 并上传文件

3. 双打包方式支持
   - PyInstaller 版本: 兼容性优秀，打包速度快
   - Nuitka 版本: 性能优秀，启动速度快
   - 两个版本均为 Standalone 模式（非单文件）

4. 完善项目文档
   - 添加 README.md 项目说明
   - 添加 .gitignore 文件
   - 优化依赖管理（requirements-minimal.txt）

【技术改进】
- 使用 Git 进行版本管理
- 自动化构建和发布流程
- 支持语义化版本号（v1.5.0）
